# Computational Statistics: Generalized Crack Distribution

This folder contains the computational portion of my generalized crack distribution work. My contribution focused on implementing and testing the numerical/statistical procedures, not claiming ownership of the full theoretical project direction.

## Project Scope

- Generalized crack random-number generation
- Density evaluation for the generalized crack distribution
- Maximum likelihood estimation
- Method of moments estimation
- EM-style parameter estimation
- Goodness-of-fit testing
- Simulation-based comparison of estimation behavior

## Included Work

- [parameter_estimation.pdf](parameter_estimation.pdf) - rendered parameter-estimation report
- [GCR&ParameterEstimation.Rmd](GCR&ParameterEstimation.Rmd) - R Markdown source for the parameter-estimation analysis
- [Estimation procedures Generalized Crack.docx](Estimation%20procedures%20Generalized%20Crack.docx) - estimation procedure notes
- [Method of Moments.docx](Method%20of%20Moments.docx) - method of moments notes
- [Goodness of Fit test.docx](Goodness%20of%20Fit%20test.docx) - goodness-of-fit notes

## Technical Summary

The main computational work translates distributional definitions into executable R analysis. The report compares estimation strategies through repeated simulation, checks parameter recovery, visualizes estimator distributions, and compares fitted densities against the true generating density.
